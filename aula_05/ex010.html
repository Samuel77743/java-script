Iniciando estudos com DOM(Document Object Model)

O DOM é um conjunto hierárquico de objetos componentes dos elementos que um navegador elabora em determinado site. Ou seja existem elementos pais e filhos, sendo o objeto "window" o raiz.

Com pleno conhecimento do DOM, é possível selecionar tags, e manipular conteúdos.
